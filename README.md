# Сайт психолога — Мельник Дарья Сергеевна

Одностраничный сайт-визитка: главная с фото, слайдер сертификатов, услуги и расценки, форма записи с уведомлением в Telegram.

## Структура проекта

```
Психолог/
├── index.html          # Главная страница
├── css/
│   └── style.css       # Стили
├── js/
│   └── main.js         # Слайдер, меню, форма
├── images/             # Фото и сертификаты (добавьте свои)
│   ├── photo.jpg       # Главное фото психолога
│   ├── cert-1.jpg      # Сертификат 1
│   ├── cert-2.jpg      # Сертификат 2
│   ├── cert-3.jpg      # Сертификат 3
│   └── cert-4.jpg      # Сертификат 4
├── netlify/
│   └── functions/
│       └── telegram-notify.js   # Отправка заявок в Telegram
├── netlify.toml        # Настройки Netlify
└── README.md
```

## Изображения

Положите в папку `images/`:

- **photo.jpg** — ваше основное фото (рекомендуемый размер около 480×600 px).
- **cert-1.jpg**, **cert-2.jpg**, **cert-3.jpg**, **cert-4.jpg** — фото сертификатов/дипломов (можно меньше карточек — отредактируйте блок с классом `cert-slider__track` в `index.html`, удалив лишние `.cert-card`).

Если нужны другие имена файлов — измените атрибуты `src` у соответствующих тегов `<img>` в `index.html`.

## Настройка уведомлений в Telegram

Чтобы заявки с формы приходили вам в Telegram:

### 1. Создать бота

1. Откройте Telegram, найдите **@BotFather**.
2. Отправьте команду `/newbot`, придумайте имя и username бота.
3. BotFather пришлёт **токен** вида `123456789:ABCdefGHI...` — сохраните его (это `TELEGRAM_BOT_TOKEN`).

### 2. Узнать свой Chat ID

1. Напишите вашему боту любое сообщение (например, «Привет»).
2. Откройте в браузере (подставьте свой токен):
   ```
   https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates
   ```
3. В ответе найдите `"chat":{"id": 123456789` — число `123456789` и есть ваш **Chat ID** (`TELEGRAM_CHAT_ID`).

### 3. Переменные окружения на Netlify

При деплое на Netlify:

1. Зайдите в **Site settings → Environment variables**.
2. Добавьте:
   - **TELEGRAM_BOT_TOKEN** — токен от BotFather.
   - **TELEGRAM_CHAT_ID** — ваш Chat ID (число, без кавычек).

После сохранения пересоберите сайт (Deploys → Trigger deploy → Deploy site).

## Локальный запуск

Просто откройте `index.html` в браузере или используйте локальный сервер:

```bash
# Из папки проекта (Психолог)
npx serve .
```

Форма с уведомлением в Telegram будет работать только после деплоя на Netlify (или другого хостинга с поддержкой serverless-функций и заданными переменными).

## Деплой в интернет (Netlify)

1. Зарегистрируйтесь на [netlify.com](https://www.netlify.com).
2. **Drag & drop**: перетащите папку проекта (всю папку «Психолог») в зону деплоя на [app.netlify.com](https://app.netlify.com) — сайт соберётся и получит адрес вида `random-name.netlify.app`.
3. Либо через Git:
   - Создайте репозиторий (GitHub/GitLab), загрузите туда проект.
   - В Netlify: **Add new site → Import an existing project** → выберите репозиторий.
   - **Build settings**: Publish directory — `.` (корень), Functions directory — `netlify/functions` (часто подставляется автоматически из `netlify.toml`).
4. В **Site settings → Environment variables** добавьте `TELEGRAM_BOT_TOKEN` и `TELEGRAM_CHAT_ID` (см. выше).
5. Сделайте **Trigger deploy**, чтобы применить переменные.

Готово: сайт в сети, заявки приходят в Telegram.

## Альтернатива: хостинг без функций (только HTML)

Если хотите выложить сайт без Netlify (например, на GitHub Pages или обычный хостинг):

- Форму можно переключить на сервис вроде [Formspree](https://formspree.io): заменить `action` формы на их URL — заявки будут приходить на email.
- Уведомления в Telegram в этом случае нужно настраивать отдельно (например, через Zapier или свой маленький сервер).
